---
## Category 테이블

대회의 분야(종목)를 저장하는 테이블입니다.

| 컬럼명  | 설명  |
|-------- |------|
| category_id | 종목 번호(PK) |
| category_name | 종목 이름 |

예시

|1|AI|
|2|Security|
|3|Game|
|4|Hackathon|

--------------------

## Host 테이블

대회를 주최하는 기관을 저장하는 테이블입니다.

| 컬럼명 | 설명 |
|--------|------|
| host_id | 기관 번호(PK) |
| host_name | 기관 이름 |

--------------------

## Contest 테이블

실제 대회 정보를 저장하는 핵심 테이블

| 컬럼명 | 설명 |
|--------|------|
| contest_id | 대회 번호(PK) |
| contest_name | 대회 이름 |
| category_id | 종목 번호(FK) |
| host_id | 주최기관 번호(FK) |
| deadline | 접수 마감일 |
| team_size | 참가 가능 인원 |
| url | 공식 홈페이지 |

---------------------
